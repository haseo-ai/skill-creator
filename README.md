# oc-skill-creator

OpenClaw-native skill creator with validated patterns and examples.

## Install

```bash
openclaw skills install oc-skill-creator
```

## What Makes This Different

- **OpenClaw-native patterns**: Phase 0 analysis, sub-agent context injection, Style Guide-first approach
- **5 validated types**: pipeline, single-task, tool-integration, code-generation, doc-conversion
- **Codex porting checklist**: Migrate existing Codex skills to OpenClaw patterns
- **Multilingual**: Keywords and reference docs in EN/KO/JA/ZH

## Quick Start

Just say something like:
- "Create a skill for converting currencies"
- "Make a skill that summarizes web pages"
- "Build a skill for generating project scaffolds"

The skill-creator will guide you through a 5-step process:
1. Analyze good and failed results
2. Write Style Guide (judgment criteria)
3. Structure workflow (Phase 0 → execution → validation)
4. Split references if needed
5. Validate with oc-skill-audit
