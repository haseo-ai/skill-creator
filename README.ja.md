# oc-skill-creator

検証済みパターンと例を備えたOpenClawネイティブスキルクリエイター。

## インストール

```bash
openclaw skills install oc-skill-creator
```

## 特徴

- **OpenClawネイティブパターン**: Phase 0分析、sub-agentコンテキスト注入、Style Guideファーストアプローチ
- **5つの検証済みタイプ**: pipeline, single-task, tool-integration, code-generation, doc-conversion
- **Codex移植チェックリスト**: 既存のCodexスキルをOpenClawパターンに移行
- **多言語対応**: EN/KO/JA/ZHのキーワードおよび参考ドキュメント

## クイックスタート

次のように言ってみてください：
- "通貨変換のスキルを作って"
- "ウェブページを要約するスキルを作って"
- "プロジェクトスキャフォールドを生成するスキルを作って"

skill-creatorが5つのステップでガイドします：
1. 良い結果と失敗例の分析
2. Style Guideの作成（判断基準）
3. ワークフローの構造化（Phase 0 → 実行 → 検証）
4. 必要に応じて参照資料を分割
5. oc-skill-auditで検証
