# oc-skill-creator

具备经过验证的模式和示例的 OpenClaw 原生技能创建器。

## 安装

```bash
openclaw skills install oc-skill-creator
```

## 特色

- **OpenClaw 原生模式**: Phase 0 分析、sub-agent 上下文注入、Style Guide 优先方法
- **5 种验证类型**: pipeline、single-task、tool-integration、code-generation、doc-conversion
- **Codex 迁移检查清单**: 将现有 Codex 技能迁移至 OpenClaw 模式
- **多语言支持**: EN/KO/JA/ZH 关键词及参考文档

## 快速开始

只需这样说：
- "创建一个货币转换技能"
- "创建一个网页摘要技能"
- "创建一个项目脚手架生成技能"

skill-creator 将引导您完成 5 步流程：
1. 分析成功案例与失败案例
2. 编写 Style Guide（判断标准）
3. 构建工作流（Phase 0 → 执行 → 验证）
4. 按需拆分参考文档
5. 使用 oc-skill-audit 进行验证
